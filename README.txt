# Web Scraping Bot

-Este é um bot desenvolvido em Python que utiliza técnicas de web scraping para coletar informações do site IBGE referente a dados dos estados Brasileiros.

# Funcionalidades

-Coleta informações de sites e as armazena em um arquivo Excel.
-Envia um email com o arquivo Excel anexado ao final do processo de coleta.
-Possui opções para personalização do REMETENTE e DESTINATÁRIO do e-mail.

# Dependências

-Python 3
-Bibliotecas: email.mime.application, MIMEApplication, botcity.web, openpyxl, smtplib, EmailMessage, Workbook

# Personalização

-Para alterar o REMETENTE e DESTINATÁRIO é necessário abrir o arquivo "./bot.py" com o seu IDE de preferência, e localizar a partir da linha: 134

# Limitações

-Este bot é desenvolvido para fins educacionais e não é recomendado para coletar informações em massa de sites.
-Uma vez que isso pode violar os termos de serviço dos mesmos e configurar práticas ilegais.
-O uso deste bot deve ser limitado a situações em que a coleta de informações é permitida ou autorizada pelo proprietário do site.
